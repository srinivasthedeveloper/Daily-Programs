Replace X - Divisible by 4

A number N with its last digit masked as X is passed as the input to the program. The program must print all the possible numbers which are divisible by 4 by replacing X in N in ascending order as the output.

Boundary Condition(s):
10 <= N <= 10^18

Input Format:
The first line contains N.

Output Format:
The first line contains the numbers based on the given condition.

Example Input/Output 1:
Input:
6447X

Output:
64472 64476

Explanation:
Here N = 6447X.
The two possible numbers divisible by 4 are 64472 and 64476.

Example Input/Output 2:
Input:
246X

Output:
2460 2464 2468
